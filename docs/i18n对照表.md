# ExModem 多语言字符串对照表

本表是 ExModem 全部界面 / 运行时字符串的本地化主表,也是生成 5 个 resx 资源文件的**唯一真相源**。

- 支持语言:**简体中文 (zh-CN)**、**繁體中文 (zh-Hant)**、**English (en, 中性/回退)**、**Русский (ru)**、**日本語 (ja)**。
- `Resources.resx` = 英文(中性回退);`Resources.<culture>.resx` = 对应语言卫星资源。
- `{0} {1}` 是 `string.Format` 占位符,各语言必须保留同样的占位符,顺序可按语法调整。
- 末尾 `…`、`✓`、`✗`、`·` 等符号原样保留。
- **审查重点**:① ru / ja 译文是否自然、术语是否准确(LTE/5G/SIM/SMS/modem 等保留英文缩写);② 占位符是否完整;③ 是否有语气/长度问题(按钮文字宜短)。

> 给审查 AI:请逐行核对 ru、ja 两列(zh-Hant 也请扫一遍),指出不自然或错误的译法并给出修订建议。简中是原文基准,英文已人工校。
